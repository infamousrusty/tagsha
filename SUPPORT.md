# Support

## How to get help

| Channel | Purpose |
|---|---|
| [GitHub Issues](https://github.com/infamousrusty/tagsha/issues) | Bug reports and feature requests |
| [GitHub Discussions](https://github.com/infamousrusty/tagsha/discussions) | Questions, ideas, and community chat |
| [Security Advisories](https://github.com/infamousrusty/tagsha/security/advisories/new) | Responsible vulnerability disclosure |

## Before opening an issue

1. Check the [README](README.md) and [docs/](docs/) for existing answers
2. Search [open and closed issues](https://github.com/infamousrusty/tagsha/issues?q=is%3Aissue) for duplicates
3. If it is a security vulnerability, follow [SECURITY.md](SECURITY.md) — do **not** open a public issue

## Response times

This is a maintainer-supported open source project. Response times are best-effort. Security issues are prioritised.
